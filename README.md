# ReMarNet
Code release for the paper [ReMarNet Conjoint Relation and Margin Learningfor Small-Sample Image Classification](#).

## Dataset
#### UIUC_Sports
You can download the data  at http://vision.stanford.edu/lijiali/event_dataset/

## Requirements
* python=2.7
* PyTorch=1.4.0
* torchvision=0.5.0
* pillow=6.2.1
* numpy=1.15.4
